<h1> Nice Data Visualization </h1>

<p> This project was created in Visual Studio Code utilizing Python 3.10.5 </p>

<h2> How to run </h2>

<ol>
  <li> Download source code </li>
  <li> Install required Python packages in requirements.txt into a Python virtual environment using pip </li>
  <li> Run gui.py </li>
</ol>

<h2> Inputs </h2>

![Default screen](/images/default_screen.png)

<ul> 
  <li>Open Input File: This will open a system file explorer for the user to input a .csv file.</li>
    <ul> 
      <li> NOTE: Check the "test-files/" folder to see how .csv files should be formatted. </li>
    </ul>
  <li>Enter Image Name: This specifies the filename of the output data timeline image file.</li>
    <ul> 
      <li> NOTE: For the current implementation, the user must also input their desired file format (i.e. .png, .jpg, etc) </li>
    </ul>
  <li>Enter Timeline Name: This specifies the title of the timeline in the image output file.</li>
</ul>
